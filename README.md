# file-system-FAT16X
使用一个固定2G大小的文件模拟磁盘，实现文件系统FAT16X，并实现简易的shell控制台以进行文件操作。

| 命令  | 说明   |
| ---- | -----  |
| format     | 格式化磁盘 |
| mkdir      | 创建目录   |
| touch      | 创建文件   |
| ls         | 显示当前目录下的所有文件和子目录 |
| cd         | 切换目录   |
| pwd        | 查看当前路径   |
| echo       | 回显内容   |
| cat        | 显示文件内容   |
